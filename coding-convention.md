## Git
### commit message
|type||
|------|---|
|feat|기능 추가|
|refactor|코드 수정|
|docs|문서 수정|
|fix|버그 수정|
|test|테스트 코드|
|chore|그 외 변경 사항|

- ex) feat: 기능 추가
